1. Konsep Utama 1. Main Concept 1. 核心概念

Daripada mengandalkan API ShortPro.app sebagai sumber data, event system menjadi "pusat aktivitas" yang mandiri. Seluruh tracking dilakukan oleh sistem event, lalu hasilnya dibuka melalui Open API. Instead of relying on the ShortPro.app API as a data source, the event system becomes an independent "activity hub." All tracking is handled by the event system, then results are exposed through an Open API. 不依赖ShortPro.app API作为数据源，而是让活动系统成为独立的"活动中心"。所有追踪由活动系统处理，结果通过Open API对外开放。
